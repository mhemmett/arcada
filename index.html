The Challenge

Valuable data,
interdisciplinary impact

The RCA hosts continuous observational data spanning geophysics, geochemistry, biology, and oceanography — yet reliably accessing it remains a significant barrier to interdisciplinary research.

01
Inconsistent API Coverage

Seismometers and pressure gauges offer seamless access; PI-operated instruments like scanning sonar and mass spectrometers do not.

02
Fragmented HTML Storage

Many instruments store unprocessed sensor readings on standalone HTML pages with no metadata or programmatic interface.

03
Siloed Disciplines

Few researchers integrate data across domains — methane seeps, hydrothermal vents, and volcanic activity remain separately studied despite deep connections.

04
Wasted NSF Investment

Scientists invest substantial time locating and contextualizing data before any analysis, reducing the scientific return on a decade of infrastructure investment.

Capabilities

What aRCADA does

// ask
Ask a Question
Get plain-language answers about instruments, sites, and science — what a BOTPT measures, what happened at Axial in 2015, how hydrothermal venting and seismicity relate.
// literature
Literature Review
Search a curated library of RCA-related publications. Ask "what research exists on methane seeps?" and get cited summaries drawn from indexed papers.
// script
Python Script Generation
For data requests, aRCADA builds a standalone .py script pre-filled with instrument IDs, stream names, and your time range — download and run locally.
// access
Unified Instrument Coverage
Bridges API-accessible instruments (seismometers, pressure gauges, CTDs) with PI-operated portal instruments (sonar, mass spectrometers, DAS) — all in one query.
// scale
Decade-Scale Archive
Access continuous observational records spanning over ten years — methane seep activity, hydrothermal vent chemistry, and Axial Seamount eruption cycles.
// open
Open & Extensible
Built on Gemini API, Cloudflare Workers, and GitHub Pages. The RAG index, instrument catalog, and data-pull scripts are all open-source and community-extensible.

Roadmap

Future directions

Scoped initially to the OOI platform, designed from the start to expand.

01
Expanded Literature Corpus
Index full-text PDFs from the RCA research literature, moving beyond abstracts to enable paragraph-level retrieval and broader interdisciplinary synthesis.
Near-term
02
User Persona Modes
Tailor responses for expert scientists, interdisciplinary researchers, and the general public — same data, different depth and framing.
Near-term
03
Notebook Integration
Generate Jupyter notebooks rather than bare scripts — with inline data loading, exploratory plots, and suggested analyses appropriate to the instrument type.
Phase 2
04
COSZO Integration
Adapt aRCADA as the data assistant for the forthcoming Cascadia Offshore Subduction Zone Observatory to support earthquake and tsunami early warning research.
